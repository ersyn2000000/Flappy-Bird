Flappy Bird — это простая аркадная игра, ставшая невероятно популярной благодаря своей механике и сложной, но затягивающей геймплейной концепции. В игре игрок управляет маленькой птичкой, которая должна пролетать между рядами труб, избегая столкновений.

Основная идея игры
Игроку нужно нажимать на экран (или клавиши), чтобы заставить птичку подниматься вверх. Между нажатиями птичка падает вниз из-за гравитации. Цель — набрать как можно больше очков, проходя через зазоры между трубами.

Ключевые элементы игры
Птичка

Главный персонаж игры, управляемый игроком.
Движется по вертикали: поднимается при нажатии и падает под действием гравитации.
Представлена в коде текстурой (Texture) и окружена кругом для обработки столкновений (Circle).
Трубы

Основные препятствия в игре. Расположены парами: верхняя труба и нижняя труба.
Расстояние между трубами (зазор) фиксированное.
Их положение по вертикали варьируется случайным образом (с помощью генератора случайных чисел).
Постепенно движутся влево с фиксированной скоростью.
Очки

Игрок получает 1 очко за каждую пройденную пару труб.
Отображаются на экране с помощью текста (BitmapFont).
Состояния игры
Игра имеет несколько состояний:

Ожидание: игра ещё не началась. Птичка находится в центре экрана.
В процессе: птичка летает, трубы движутся, игрок набирает очки.
Конец игры: если птичка сталкивается с трубой или падает на землю, игра завершается.
Фон

Красивый задний фон делает игру визуально приятной.
Не имеет игрового влияния, используется для эстетики.
Как работает механика Flappy Bird (на примере кода)
Гравитация и движение птички

Птичка падает вниз из-за постоянного увеличения скорости падения:

velocity = velocity + gravity;
birdY -= velocity;
При нажатии на экран птичка поднимается:

if (Gdx.input.justTouched()) {
    velocity = -30;
}
Генерация труб

Положение труб определяется случайным образом:

tubeOffset[i] = (randomGenerator.nextFloat() - 0.5f) * (Gdx.graphics.getHeight() - gap - 200);
Трубы перемещаются влево, создавая иллюзию движения:
java
Копировать код
tubeX[i] = tubeX[i] - tubeVelocity;
Столкновения

Для проверки столкновений используется класс Intersector:

if (Intersector.overlaps(birdCircle, topTubeRectangles[i]) || 
    Intersector.overlaps(birdCircle, bottomTubeRectangles[i])) {
    gameState = 2; // Конец игры
}
Перезапуск игры

После окончания игры, при нажатии игра начинается заново:


if (gameState == 2 && Gdx.input.justTouched()) {
    gameState = 1;
    startGame();
    score = 0;
    velocity = 0;
}
Отображение очков

Очки отображаются с помощью шрифта:

font.draw(batch, String.valueOf(score), 100, 200);
Особенности реализации в LibGDX
SpriteBatch используется для отрисовки всех элементов игры (фон, птичка, трубы, текст).
ShapeRenderer и Intersector применяются для обработки столкновений.
Texture отвечает за загрузку и отображение изображений.
Random используется для генерации случайного положения труб.
Почему Flappy Bird так популярна?
Простота управления
Нажатие одной кнопки — это всё, что нужно, чтобы играть.

Высокая сложность
Лёгкость механики сочетается с высокой сложностью прохождения, что вызывает желание попробовать ещё раз.

Минималистичный дизайн
Простая, но приятная графика, подходящая для всех возрастов.

Затягивающий процесс
Каждое достижение мотивирует побить свой рекорд.
